EXPERIMENT NO:3

CODE CONVERTERS

AIM:

To design and setup the following circuits:

1. A 4-bit Binary to Gray converter

2. A 4-bit BCD to Excess-3 converter

COMPONENTS AND EQUIPMENTS REQUIRED:

IC-7408, IC-7432, IC-7404, IC-7486, IC-Trainer kit

THEORY:

To convert a Binary number to corresponding Gray code, the following rules are applied:

1. The MSB in the Gray code is the same as the corresponding bit in a Binary number.

2. Going from left to right, add each adjacent pair of binary digits to get the next Gray code. Discard carries.

As the first step to design a binary to Gray code converter, set up a truth table with binary numbers B3 B2 B1 B0 and corresponding Gray code numbers G3 G2 G1 G0. Set up a circuit realizing the simplified logic expression obtained using K maps for Gs as the number of Bs.

To convert BCD to corresponding Excess-3 code, the following rules are applied:

1. Group the Binary digits into sets of 4, then add 0011 to each group. This addition of 0011 to each groups of 4 bits converts BCD into Excess-3 code.

2. If the sum of the original binary digit and 0011 exceeds 9, a carry is generated and added to the next group.

3. Repeat the process for each group of 4 bits until the entire binary number is converted into Excess-3.

As the first step to design BCD to Excess-3 converter, set up a truth table with BCD numbers A, B, C, D and corresponding Excess-3 as W,X,Y,Z. Set up a circuit realizing the simplified logic expression obtained using K maps.

PROCEDURE:

1. Place the IC on IC trainer kit

2. Convert Vcc and GND to respective pins of IC trainer kit

3. Connect the inputs to the input switches provided in the IC trainer kit

4. Connect the outputs to the switches of output LEDs

5. Apply various combination of inputs according to the truth table and observe condition of LEDs

6. Disconnect output from the LEDs and note down the corresponding Multimeter Voltage reading for various combination of inputs

RESULT:

Binary to Gray Converter and BCD to Excess-3 converter are designed and truth table is verified.
