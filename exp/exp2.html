EXPERIMENT NO:2

HALF AND FULL ADDER & HALF AND FULL SUBTRACTER

AIM:

To implement the half adder, full adder, half subtracter and full subtracter circuit using basic gates,

universal gates and verify their truth table.

COMPONENTS AND EQUIPMENTS REQUIRED:

IC 7486, 7432, 7400, 7404, 7408, IC trainer kit.

THEORY:

The simplest binary adder is half adder. Half adder has 2 input bits and two output bits. One output

bit is the sum and other is carry. They are represented by S and C respectively in their logical symbol.

A half adder has no provision to add a carry from lower order bit when binary numbers are added. When 2

input bits and a carry to be added, the number of input bits become 3 and input combination increase to

8. For this, a full adder is used. Like half adder it also has a sum bit and carry bit. The new carry

generated is represented by C and carry generated by C from the previous addition is represented by Cn-1.

A logical circuit for the binary subtraction between two single bit binary numbers is known as half

subtracter. It consist of 2 input bits and output bits. One output bit is borrow and other output is

difference. They represented by B' and D respectively in logic symbols. From the truth table it is clear

that the difference output is zero if A=B and 1 if A!=B.

by D and B' which is difference and borrow respectively.

PROCEDURE:

1. Verify whether all components and wires are in good conditions

2. Set up the half adder circuit and feed the input combinations

3. Observe the output corresponding to input combination and enter it in the truth table

4. Repeat the above steps for a full adder

The half adder, full adder, half subtractor, full subtractor circuits are implemented using basic gates
